Site Description

This site is designed to allow everyday users to find suppliers of free food and to also connect volunteer suppliers to these users. Think of a college student trying to find a food pantry near them, they would simply input their zip code (30303) in the search bar and would be directed to a list of options near them with the option of creating a route to their destination Currently this site is only able to use the 30303 or 30097 zip code due to time constraints of the project.
