About Us Page

This project was chosen to offer accessible, free service for those in need looking for food from volunteering suppliers. For this project technologies such as php, html, css, mySQL database, phpMyAdmin, javascript, and google maps were used. In this class, we learned how to incorporate elements of html, css, php, and JavaScript to create a website in real time that can help locate food sources through zip code.
